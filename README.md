blp-bcpc-cookbook
=================
Configures nodes to deploy the [Bloomberg Clustered Private Cloud][1].

This is a refactoring of [the original cookbook][1] which utilizes
[test-kitchen][2], [Berkshelf][3] and community cookbooks for
configuration.

# Requirements

## Laptop Testing
- [Vagrant][4]
- [VirtualBox][5]

## Operating System
- [CentOS 6.5][6]
- [Ubuntu 12.04][7]
